# 🏥 Doctor Landing Page (High-Converting + Premium UI)

A modern, responsive **doctor landing page** built using **Next.js + React**, designed to maximize **patient bookings and trust**.

This project includes **two design variations**:

* ⚡ Conversion-focused UI (lead generation)
* 🎨 Premium brand-focused UI (luxury medical experience)

## 🛠️ Getting Started

Follow these steps to run the project locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

---

### 2️⃣ Install dependencies

```bash
npm install
```

---

### 3️⃣ Run the development server

```bash
npm run dev
```

---

### 4️⃣ Open in browser

```
http://localhost:3000
```

---

## ⚡ Notes

* Make sure you have **Node.js (v16 or higher)** installed
* If you face issues, delete `node_modules` and run:

```bash
npm install
```
